# Discovery Server
스프링 클라우드는 Discovery Server에 벤더 독립적 접근 방식을 제공한다. 이번 프로젝트에서는 여러 구현체 중 Neflix Eureka를 사용한다.

Spring Cloud为Discovery Server提供了独立于供应商的方法。 在该项目中，Neflix Eureka被用于多种实现中。
